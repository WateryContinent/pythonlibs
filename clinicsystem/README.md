# Python Clinic System
A simple CLI Python-based clinic management system for managing staff accounts, working hours, and patient records. Written for assessment, can be useful as reference material if needed.

---

## Features

- Create Doctor or Nurse accounts
- Secure login system (with encryption)
- Log staff working hours
- Nurses can log patients attended
- View registered users
- Simple command-based interface
- Basic GUI window (Tkinter)
